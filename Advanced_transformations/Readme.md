| Transformation Type | Purpose |
|---|---|
| MERGE INTO (Upsert) | Update existing records and insert new records |
| Change Data Capture (CDC) | Track INSERT, UPDATE, and DELETE changes incrementally |
| Window Functions | Perform ranking, running totals, and row-based calculations |
| OPTIMIZE | Compact small files into larger files for better performance |
| ZORDER Clustering | Improve query performance using data skipping |
| Partitioning | Split data into partitions for faster data scanning |
| Liquid Clustering | Dynamically organize data without fixed partitions |
| Time Travel | Access previous versions of Delta tables |
| VACUUM | Remove old unused files from storage |
| Schema Evolution | Handle schema changes automatically |
| Streaming Transformations | Process real-time streaming data |
| Data Deduplication | Remove duplicate records from datasets |
